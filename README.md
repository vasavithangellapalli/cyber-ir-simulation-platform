# Cyber Incident Response Simulation Platform

## Overview

The Cyber Incident Response Simulation Platform is a web-based cybersecurity training and investigation platform designed to simulate real-world security incidents. It helps users practice incident detection, investigation, threat analysis, and response activities in a controlled environment.

## Features

* Incident Response Simulation
* Threat Analysis Dashboard
* Security Event Investigation
* Network Activity Monitoring
* Log Analysis
* Response Action Management
* Incident Reporting
* Performance Evaluation and Scoring

## Technologies Used

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js
